Order::sendEmail

Send the invoice of the order by email.

    Signature

    public function sendEmail([string $subject = "", [String $message = "", [Boolean $resend = false, [Boolean $adminOnly = false, [string $emailClassName = 'Order_InvoiceEmail']]]]] )

    Parameters

    $subject — string
    - subject for the email


    $message — string
    - the main message in the email


    $resend — object
    - send the email even if it has been sent before


    $adminOnly — object
    - do not send to customer, only send to shop admin


    $emailClassName — string
    - class used to send email
