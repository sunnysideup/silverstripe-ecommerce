Order::sendAdminNotification

Sends a message to the shop admin ONLY and not to the customer This can be used by ordersteps and orderlogs to notify the admin of any potential problems.

    Signature

    public function sendAdminNotification([string $subject = "", [string $message = ""]] )

    Parameters

    $subject — string
    - subject for the email


    $message — string
    - message to be added with the email


    Returns

    Boolean
    TRUE for success, FALSE for failure (not tested)
